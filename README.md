<h1>Video Game Roll-a-ball</h1>
<h3>INTRODUCCIÓN</h3>
<p> Este proyecto nace de la necesidad de desarrollar un videojuego, con fines interactivos y de aprendizaje 
en la elaboración y programación del mismo y poder incursionar en el mundo Gamer.</p>
<h3>ANALISIS</h3>
<p>Para este ejercicio se ha pensado en la creación de una mesa parecida al de billar,donde el jugador representado por una bola atraviesa una gama de obstaculos para superar, con la intención de sumar puntos, antes de ser atrapado por un enemigo.</p>
<h1>PRE-REQUISITOS</h1>
<P>-UNITY: -'https://unity.com/es/download' - Git: 'https://git-scm.com/downloads -Visual Studio Community 2022 'https://visualstudio.microsoft.com/es/vs/community/'</P>
<h1>#Instalación:</h1>
<p># Clonar el repositorio 'https://github.com/joelc74/Roll-a-ball.git'</p>
<h1>Configurar Unity</h1>
<p>1- Una vez instalado Unity pasamos a crear Unity ID donde introducimos un email y un password para crear nuestra cuenta.</p>
<p>2- Instalamos Unity Editor preferiblemente última versión LTS</p>
<p>3- Creamos el Proyecto al que denomine Roll-a-ball en 3D (Built-in Render Pipeline)</p>
